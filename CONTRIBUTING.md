# Contributing to Kunaal's Illustrations

Thanks for your interest in contributing! This is an open source Agent Skill — contributions that improve the style rules, character IPs, composition patterns, or documentation are welcome.

## What You Can Contribute

- **Style improvements** — refining `style-dna.md`, `character-ip.md`, or `composition-patterns.md`
- **Prompt template improvements** — better or more reliable prompt structures in `prompt-template.md`
- **QA checklist additions** — new checks that catch common generation failures
- **Example images** — new hand-drawn illustrations that fit the skill's visual language
- **Bug fixes** — corrections to incorrect or conflicting rules across reference files
- **Documentation** — README improvements, clearer instructions, or new usage examples

## What to Avoid

- Adding new visual styles that conflict with the existing hand-drawn, white background aesthetic
- Adding commercial or complex illustration types (PPT-style, infographics, etc.)
- Changing the Reference Boy or Girl IP appearance without discussion first

## How to Contribute

1. Fork the repository
2. Create a branch: `git checkout -b your-branch-name`
3. Make your changes
4. Open a Pull Request — use the PR template and describe what you changed and why
5. The maintainer will review and respond within a few days
