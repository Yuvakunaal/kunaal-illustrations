# Security Policy

## Supported Versions

This repository contains a skill definition (Markdown files, YAML configs, and PNG assets). There is no executable code, server, or binary — so traditional software vulnerabilities do not apply.

| Content | Maintained |
|---|---|
| SKILL.md + references | ✅ Yes |
| agents/openai.yaml | ✅ Yes |
| Example assets | ✅ Yes |

## Reporting a Vulnerability

If you discover a security concern related to this repository — such as a malicious dependency, a compromised asset, or misuse of the skill configuration — please report it privately:

**Email:** bhavikunaal@gmail.com  
**Subject line:** `[SECURITY] kunaal-illustrations — <brief description>`

Do **not** open a public GitHub issue for security concerns. You can expect an acknowledgement within 48 hours.
