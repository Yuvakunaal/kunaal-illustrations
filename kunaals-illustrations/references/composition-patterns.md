# Composition Patterns & Originality Rules

## Basic Structure Types

Choose one structure type per image; do not mix too many.

### Workflow
Suitable for: Input -> Processing -> Output, content production, AI workflows, automation chains.
How to draw: Input on the left, character or strange machine processing in the middle, output on the right, orange arrows indicating the main flow.

### System Component
Suitable for: Information sources, filters, databases, renderers, local parts of an agent system.
How to draw: Draw only 3-5 core modules. The character participates in one key action within it.

### Before & After
Suitable for: Chaos/Order, Manual/Auto, Scattered/Gathered, Anxious/Stable.
How to draw: Chaos on the left, stability on the right, orange arrow in the middle. The character can be more exaggerated.

### Role State
Suitable for: User pain points, creator states, too many tools, information anxiety, from being stuck to running smoothly.
How to draw: 2-4 small states, one short label per state.

### Conceptual Metaphor
Suitable for: Content factory, information warehouse, mental black box, workflow machine, automated daily reports.
How to draw: A large, strange object or machine, minimal inputs, one output. It must be memorable.

### Method Layering
Suitable for: Methodology frameworks, system hierarchies, capability stacks, content system layers.
How to draw: Layers of boxes (do not make a formal pyramid). The character is beside them, building or carrying bricks.

### Map Route
Suitable for: From idea to launch, user journey, content continuation path, learning route.
How to draw: A winding path, a few nodes, the character pulling a line or walking.

### Comic Storyboard
Suitable for: Failure to success, real processes, complaints, before/after tool usage.
How to draw: 2-4 small panels, each panel expressing only one action.

## Original Metaphor Generation Method

You must reinvent a metaphor from the current article every time. Do not copy old images.

### Three Steps

1. Turn an abstract concept into a physical action: stuck, leak, become heavy, sort, settle, ferment, open doors, fold, unpack, backflow.
2. Turn a system structure into a low-tech object: broken machine, cardboard box, drawer, water pipe, mailbox, weird dial, scale, well, ladder, strange desk setup.
3. Let the character undertake the action: Do not just stand beside it. Get stuck in the machine, pull the wrong wire, guard the door, carry, repair, weigh, hold the ladder, record, or stuff things into a strange device.

### Available Object Pool

- Cardboard box, drawer, old machine, funnel, scale, mailbox, door, well, ladder, water pipe, ball of yarn, gate, dial, black box, hole puncher, pasta maker, clothesline, weird desk setup.
- Only pick 1-2 when generating; do not overcrowd.

### Character Action Pool

- Pull, carry on shoulder, stuff, scoop, press, weigh, sew, cut, twist, guard, push, catch, dismantle, mark, recycle.
- The action must serve the core meaning; don't make it weird just for the sake of being weird.

## Anti-Repetition Rules

Do not default to opening or replicating `assets/examples/`. These images are only used for style calibration (e.g., line density, white space, color restraint, character vibe).

Unless the user explicitly says "copy this one / replicate this composition / use this case to modify", do not directly reuse these old compositions:
- Conveyor belt with two breakpoints
- Pulling the judgment lever in the content machine
- Turning into a funnel to sort traffic/trust/conversion
- Slicing a "material fish"
- Pulling the continuation path
- Pulling three layers of information sources
- Three characters holding megaphones/building bridges/opening doors
- Stamping the script toolbox
- Holding a warning sign looking at the pitfall path

Even for similar themes, you must change the metaphor. For example, "continuation path" doesn't necessarily mean drawing a route; it could be the character connecting the tail of content to a door handle. "One fish, many uses" doesn't have to be a fish; it could be the character pressing a ball of paper into several shapes.
